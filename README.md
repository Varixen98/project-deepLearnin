<h1>Brain Tumor Detection</h1>
<p>
This is a university project for deep learning course.<br>
The model consist of YOLO11nano + Bi-directional Feature Pyramid Network (BiFPN) + Spatial Pyramid Pooling Fast (SPPF).
The dataset that were used to train the model is from:
<a href="https://arxiv.org/abs/2506.14318">BRISC: Annotated Dataset for Brain Tumor Segmentation and Classification</a>
For refference, article:
<a href="https://pmc.ncbi.nlm.nih.gov/articles/PMC10453020/#B64-cancers-15-04172">Brain Tumor Detection Based on Deep Learning Approaches and Magnetic Resonance Imaging</a>
</p>
